## Fastify 基础

### 什么是 Fastify
Fastify 是一个高性能的 Node.js Web 框架，专注于提供最佳的开发体验和最小的性能开销。

### 项目初始化
```json
{
  "type": "module",  // 使用 ES Module
  "dependencies": {
    "fastify": "^4.26.2",
    "@fastify/mysql": "^4.0.0",
    "fastify-plugin": "^5.1.0"
  }
}
```

### 基本服务器搭建
```javascript
import fastify from 'fastify'

const app = fastify()

app.listen({ port: 3000 }, (err, address) => {
    if (err) {
        console.error(err)
        process.exit(1)
    }
    console.log(`Server listening at ${address}`)
})
```

## 路由处理

### GET 请求
```javascript
// 方式1: 使用 res.send()
app.get('/', (req, res) => {
    console.log(req.query)  // 获取查询参数
    res.send({ message: 'Hello World!' })
})

// 方式2: 直接 return（推荐）
app.get('/', (req, res) => {
    return { message: 'Hello World!' }
})
```

**知识点：**
- Fastify 没有 `.json()` 方法，使用 `.send()` 或直接 `return`
- `req.query` 获取 URL 查询参数

### POST 请求
```javascript
app.post('/login', (req, res) => {
    console.log(req.body)  // 获取请求体
    res.send({ message: 'Login successful!' })
})
```

**知识点：**
- Fastify 天然支持 POST 请求的 JSON 解析
- `req.body` 直接获取请求体数据

### 路由参数
```javascript
app.get('/user/:id', (req, reply) => {
    const userId = req.params.id  // 获取路由参数
    // 处理逻辑...
})
```

### 使用 route 方法定义路由
```javascript
app.route({
    url: '/add',
    method: 'POST',
    handler: (request, reply) => {
        return {
            message: 'success',
            data: [{
                a: request.body.a,
                b: request.body.b,
                result: request.body.a + request.body.b
            }]
        }
    }
})
```

## 数据库集成

### MySQL 连接配置
```javascript
import mysql from '@fastify/mysql'

app.register(mysql, {
    promise: true,  // 使用 Promise 模式
    connectionString: 'mysql://root:123456@localhost:3306/xiaomi'
})
```

### 测试数据库连接
```javascript
app.get('/test', async (req, reply) => {
    try {
        const connection = await app.mysql.getConnection()
        const [rows] = await connection.query('SELECT 1 as test')
        connection.release()  // 释放连接
        reply.send({ status: 'ok', result: rows })
    } catch (err) {
        reply.status(500).send({ error: err.message })
    }
})
```

### 查询数据（带参数）
```javascript
app.get('/user/:id', (req, reply) => {
    app.mysql.getConnection(onConnect)

    async function onConnect(err, client) {
        if (err) return reply.send(err)
        
        try {
            const result = await client.query(
                'SELECT id, username, hash, salt FROM user WHERE id=?',
                [req.params.id]  // 防止 SQL 注入
            )
            reply.send(result)
        } finally {
            client.release()  // 确保释放连接
        }
    }
})
```

### 插入数据
```javascript
app.post('/add', async (request, reply) => {
    try {
        const connection = await app.mysql.getConnection()
        const { name, hobby } = request.body
        
        try {
            const [result] = await connection.query(
                "INSERT INTO user(create_time, name, hobby) VALUES(?, ?, ?)", 
                [new Date(), name, hobby]
            )
            reply.send({ 
                message: 'success', 
                insertId: result.insertId  // 获取插入的 ID
            })
        } finally {
            connection.release()
        }
    } catch (err) {
        reply.status(500).send({ error: err.message })
    }
})
```

### 查询列表
```javascript
app.get('/list', async (request, reply) => {
    try {
        const connection = await app.mysql.getConnection()
        const [result] = await connection.query("SELECT * FROM user")
        reply.send({ result })
    } catch (err) {
        reply.status(500).send({ error: err.message })
    } finally {
        connection.release()
    }
})
```

**重要知识点：**
- 使用 `?` 占位符防止 SQL 注入
- 必须使用 `connection.release()` 释放连接
- 使用 `try...finally` 确保连接被释放
- `result.insertId` 获取插入记录的 ID

## 插件系统

### 创建插件
```javascript
import fp from 'fastify-plugin'

const fn = fp((fastify, opts, done) => {
    // 给 fastify 实例添加方法
    fastify.decorate('add', (a, b) => a + b)
    done()  // done === next，表示插件加载完成
})

app.register(fn)
```

### 使用插件
```javascript
app.post('/add', (request, reply) => {
    return {
        message: 'success',
        data: [{
            a: request.body.a,
            b: request.body.b,
            result: app.add(request.body.a, request.body.b)  // 使用插件方法
        }]
    }
})
```

**知识点：**
- `fastify-plugin` 用于创建可复用的插件
- `decorate` 方法给 fastify 实例添加自定义方法
- `done()` 回调表示插件加载完成（类似 next）

## Schema 验证

### 定义请求和响应 Schema
```javascript
app.route({
    url: '/add',
    method: 'POST',
    handler: (request, reply) => {
        return {
            message: 'success',
            data: [{
                a: request.body.a,
                b: request.body.b,
                result: request.body.a + request.body.b
            }]
        }
    },
    // 定义请求体结构
    schema: {
        body: {
            type: 'object',
            properties: {
                a: { type: 'number' },
                b: { type: 'number' }
            }
        }
    },
    // 定义响应结构
    response: {
        200: {
            type: 'object',
            properties: {
                message: { type: 'string' },
                data: {
                    type: 'array',
                    items: {
                        type: 'object',
                        properties: {
                            a: { type: 'number' },
                            b: { type: 'number' },
                            result: { type: 'number' }
                        }
                    }
                }
            }
        }
    }
})
```

**知识点：**
- Schema 用于序列化和验证数据
- `schema.body` 定义请求体结构
- `response[状态码]` 定义响应结构
- 使用 JSON Schema 格式
- Schema 可以提高性能（序列化更快）

## 常用技巧

### 启动服务器的两种方式
```javascript
// 方式1: 回调函数
app.listen({ port: 3000 }, (err, address) => {
    if (err) throw err
    console.log(`Server listening at ${address}`)
})

// 方式2: Promise
app.listen({ port: 3000 }).then(() => {
    console.log('server is running on port: 3000')
})
```

### 错误处理
```javascript
try {
    // 数据库操作
} catch (err) {
    console.error('Error:', err)
    reply.status(500).send({ error: err.message })
}
```
