## 16_spider 环境搭建与运行笔记（uv + Node）

### 安装 uv（Windows）
- **安装**

```powershell
pip install -U uv
```

- **验证**

```powershell
uv --version
```

> 如果你机器没有 pip/py：先安装 Python（勾选 Add Python to PATH），再开一个新的 PowerShell。

---

### 使用 uv 创建虚拟环境
- **在项目根目录创建 venv（会生成 `.venv/`）**

```powershell
uv venv
```

- **激活 venv（PowerShell）**

```powershell
.\.venv\Scripts\Activate.ps1
```

- **退出 venv**

```powershell
deactivate
```

---

### 安装 Python 依赖
本项目 Python 依赖写在 `requirements.txt`（含 `jieba`、`wordcloud` 等）。

- **用 uv 安装（推荐）**

```powershell
uv pip install -r .\requirements.txt
```

- **安装单个包**

```powershell
uv pip install 包名
```

---

### 安装 Node 依赖
本项目 Node 侧主要依赖 `puppeteer`。

```powershell
npm i
```

---

### 运行逻辑（我当时做了什么）
- **Node 入口**：`index.js`
  - 用 `puppeteer` 打开 `https://juejin.cn/`
  - 读取左侧菜单，点击指定分类（默认“前端”，也可传参）
  - 抓取文章标题列表
  - 使用 `spawn()` 调用 Python：`index.py <title1> <title2> ...`

- **Python 脚本**：`index.py`
  - 接收命令行传入的标题数组
  - `jieba` 分词
  - **不做词频统计**，直接展示分词结果（打印 tokens）
  - 同时生成 `wordcloud.png`（从 tokens 拼接的文本生成）

### 常见问题与处理（我遇到过的）
- **PowerShell 不支持 `cd /d ... && ...`**
  - 用 `Set-Location "路径"`（或 `cd "路径"`）即可。

- **`ModuleNotFoundError: No module named 'jieba'`**
  - 说明 Python 没用到 `.venv` 或 `.venv` 里没装依赖：

```powershell
.\.venv\Scripts\Activate.ps1
uv pip install -r .\requirements.txt
```

- **`.venv\Scripts\python.exe: No module named pip`**
  - 给 venv 补 pip（我当时用的是这个）：

```powershell
.\.venv\Scripts\python.exe -m ensurepip --upgrade
.\.venv\Scripts\python.exe -m pip install -r .\requirements.txt
```

- **`UnicodeEncodeError: 'gbk' codec can't encode ...`（标题里有 emoji）**
  - Python 打印 tokens 时会踩到控制台编码问题  
  - 我在 `index.py` 顶部加了：
    - `sys.stdout.reconfigure(encoding='utf-8', errors='replace')`
